📝 업체(Vendor) 인보이스 등록 매뉴얼

인보이스 등록 화면 주요 필드

🚢 필수 입력 정보

  • 선박명/코드: 드롭다운에서 선택
  • 기기 분류(Kind): Main Engine, Aux Engine 등
  • 비용 구분(Category): Hull, Engine 등 자동 매핑
  • 계정번호(GLV): Kind 선택 시 자동 입력
  • 통화/견적가/실적가: 금액 입력 시 쉼표 자동 포맷
  • 작업 국가 → 항구: 국가 선택 후 항구 드롭다운 제공

📋 자동 계산 필드

  • 차액(Difference): 실적가 - 견적가 자동 산출
  • 차이율(%): 실적/견적 비율 자동 표시
  • 입력 완료 후 [제출] 클릭 → 상태가 접수로 변경
  • 제출 전까지는 입력(Draft) 상태로 임시 저장 가능
💡 핵심 포인트

업체가 [제출] 버튼을 누르는 순간 → 진행상황 페이지(결재 대기열)와 선급금 페이지에 동시에 인보이스가 활성화됩니다.
임시 저장(입력) 상태에서는 감독관에게 보이지 않으므로 안심하고 작성하세요.

👨‍🔧 감독관(Supervisor) 결재 매뉴얼

감독확인 대기열 화면

  • 업체가 제출한 인보이스가 접수 상태로 [감독확인 대기] 탭에 실시간 나열됩니다.
  • 각 인보이스 카드에는 선박명, GMS코드, GLV 계정, 기기분류, 비용구분이 한눈에 표시됩니다.
  • 견적가 대비 실적가가 +20% 초과 시 ⚠️ 견적초과 경고 뱃지가 자동으로 달립니다.
  • 동일 선박/계정으로 유사 청구가 감지되면 중복 의심 알림이 표시됩니다.

승인 절차

  • ① 상세 확인: 카드를 클릭하여 견적/실적 차이, 작업기간, 항구 정보 등 검토
  • ② 코멘트 입력: 특기사항이나 확인 내용을 메모란에 기재
  • ③ [감독확인] 클릭: 상태가 감독확인으로 변경 → 팀장 대기열로 이관
⚠️ 반려 시

검토 결과 이상이 있으면 [반려] 버튼으로 업체에 재작성을 요청할 수 있습니다.
반려 사유를 상세히 기재해주세요.

👔 팀장(Team Lead) 최종 결재 매뉴얼

팀장 결재 대기열

  • 감독관이 1차 승인 완료한 건만 [팀장확인 대기] 탭에 나열됩니다.
  • 감독관이 남긴 코멘트 및 승인 일시를 확인할 수 있습니다.
  • 견적 초과 경고, 중복 의심 등의 리스크 뱃지가 그대로 유지되어 전달됩니다.

최종 결재 프로세스

  • ① 감독 코멘트 확인: 1차 검토자의 의견 열람
  • ② 금액/내역 최종 확인: 비용 적정성 판단
  • ③ [팀장 승인] 클릭: 최종 결재 완료 → 팀장승인 / 결재완료 상태

커버 출력 기능 (A4 PDF)

  • 결재 완료된 건에 대해 [인쇄/출력] 버튼 → 공식 결재 커버 페이지가 생성됩니다.
  • 인쇄 설정: A4 세로(Portrait), 상하좌우 35mm 여백 자동 적용
  • Ctrl+P 또는 브라우저 출력 → PDF로 저장 가능

💰 선급금 관리 (Advance Payment) 매뉴얼

핵심 아키텍처: 진행 상황 동기화

  • 인보이스가 [접수] 상태가 되는 즉시, 선급금 목록에 자동 활성화됩니다.
  • 임시 저장(입력) 상태의 인보이스는 목록에서 완전 배제됩니다.
  • 감독관이 100건의 인보이스 중 이번 달 정산이 필요한 50건만 체크박스(☑)로 선택 가능합니다.

🔄 환율 변환

  • 화면 상단에 KRW, EUR, JPY, SGD → USD 환율 입력란
  • 입력 즉시 모든 인보이스 금액이 실시간 USD로 자동 변환
  • 각 행에 직접 USD 금액 수기입력도 가능

📊 4-Dimension 집계

  • 선박별: 체크된 인보이스를 선박 기준 합산
  • 계정(GLV)별: 회계 계정과목별 소계
  • 비용구분별: Hull/Engine/Elect 등 카테고리별
  • 기기분류별: M/E, A/E, Gen 등 기기별 합산
📥 Excel 보고서 다운로드

체크된 인보이스들을 5개 시트(목록, 선박별, 계정별, 비용구분별, 기기분류별)로 나뉜 XLSX 파일로 즉시 다운로드합니다.

📊 대시보드 (Dashboard) 매뉴얼

실시간 비용 현황 시각화

📈 KPI 통계 카드

  • 총 인보이스 수: 전체 등록 건수
  • 총 비용: 실적가(Actual) 기준 합산
  • 결재 대기: 감독/팀장 확인 대기 건수
  • 평균 차이율: 견적 대비 실적 전체 평균 %

📊 차트 시각화

  • 월별 비용 추이: 바 차트로 월별 발생 비용 비교
  • 기기분류 분포: 가로형 바 차트로 점유율(%) 표시
  • 선박 Top 5 지출: 유지보수비 최다 선박 랭킹
  • 통화 분포: KRW, USD, EUR 비율 원형 차트
💡 대시보드 vs 설정 페이지

대시보드는 경영 데이터 시각화 전용이며, 설정(Settings) 페이지는 시스템 관리 및 보안 전용입니다.
두 페이지는 완전히 독립된 기능을 가지고 있습니다.

⚙️ 환경 설정 (Settings) 매뉴얼

🔒 보안 설정

  • 2단계 인증(2FA): 활성화/비활성화 토글
  • 자동 로그아웃: 1시간 / 4시간 / 8시간 선택
  • 데이터 보존기간: 1년 / 3년 / 5년 / 영구

🔔 알림 설정

  • 결재 요청 알림: 새 인보이스 접수 시 알림
  • 지급 완료 통보: 결재 완료 시 알림
  • 시스템 공지: 유지보수/업데이트 알림

🔗 ERP 통합 (Integration)

  • 시스템 연동: SAP / Oracle 자동 동기화 ON/OFF
  • 동기화 주기: 실시간 / 1시간 / 일 1회
  • API Key: 보안 키 재발급 및 관리

🛠️ 관리자 도구

  • 감사 로그 추출: 변경 이력 파일(JSON) 다운로드
  • Factory Reset: 전체 데이터 초기화 (확인 2회)
  • 데모 데이터: 25건 테스트 인보이스 시드

🖨️ 인보이스 커버 인쇄 매뉴얼

커버 페이지 구성

  • 인보이스 상세 조회 화면에서 [인쇄 미리보기 / Print Preview] 버튼 클릭
  • A4 규격 결재 표지가 화면 하단에 렌더링됩니다.
  • 표지에는 선박명, 작업내역, 금액(견적/실적/차액), 승인자 서명란이 포함됩니다.
